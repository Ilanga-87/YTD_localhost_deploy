About This Project

This site has been a practical application of the following skills:

— Utilizing the Django framework for Python, delving into the concept of CBVs (Class-Based Views).

— Interacting with PostgreSQL, one of the most popular databases.

— Implementing "deferred" tasks with Celery.

— Creating periodic tasks through Celery beat.

— Working with message brokers like Redis and RabbitMQ.

— Integrating third-party modules into my project.

— Bundling all components into a unified Docker compose and deploying it both locally and on a server with a domain.
